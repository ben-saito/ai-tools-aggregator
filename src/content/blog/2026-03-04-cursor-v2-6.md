Cursor released version 2.6 at the end of February. This update includes MCP Apps for interactive UI support in agent chats, team-based plugin management for Teams and Enterprise plans, and improvements to debug mode capabilities.

## MCP Apps

[MCP Apps](https://cursor.com/docs/context/mcp#mcp-apps) enable interactive user interfaces directly within agent chats. Supported tools include charts from Amplitude, diagrams from Figma, and whiteboards from tldraw.

This allows agents to display data visualization results directly in the chat or conduct UI design reviews in an interactive manner. The development workflow becomes more intuitive when communicating with agents.

## Team Plugin Management

On Teams and Enterprise plans, administrators can now create [team marketplaces](https://cursor.com/docs/plugins#team-marketplaces) to share private plugins internally. From the settings page, plugins can be managed and distributed with centralized governance and access control.

This feature enables teams to standardize team-specific workflows and best practices as plugins, making it easier for new team members to leverage them. Enterprise plans include additional granular access controls and audit capabilities.

## Debug Mode Improvements

This update also includes performance and usability improvements to debug mode. For specific details, please refer to the official documentation.
