BlockRunAIが開発したOpenClaw専用LLMルーター「ClawRouter」がリリース11日でGitHub 2,400スターを獲得し、コミュニティで急速な関心を集めている。APIキー不要のウォレット決済、100%ローカル動作の15次元スコアリングエンジン、30種以上のモデルプールという設計が、従来のLLMルーターとは一線を画す独自のアーキテクチャを形成している。

## 何が問題を解決するのか

OpenClawをClaude Opus 4.5（入力$5/出力$25 per million tokens）一本で動かすと、ハートビートや定型タスクにも最高料金が課金される。ClawRouterはリクエストの性質を判断し、適切なコスト帯のモデルに自動振り分けすることで、典型的なワークロードでOpus 4.5比**92%のコスト削減**を実現する設計だ。

## アーキテクチャの核心：15次元ローカルスコアリング

最も注目すべき技術的特徴は、分類処理が完全にローカルで完結する点だ。

```
Request → Weighted Scorer (15 dimensions)
 │
 ├── High confidence → Pick model from tier → Done
 │
 └── Low confidence → Default to MEDIUM tier → Done
```

- **判定速度**：<1ms
- **外部API呼び出し**：ゼロ
- **プライバシー**：リクエスト内容が外部に送信されない

外部の分類APIに依存するルーターと異なり、コールドスタートのレイテンシがなく、分類コストも発生しない。判定精度が低い場合はMEDIUMティアにデフォルトするフォールバック設計も実用的だ。

## 4段階のティアシステム

| ティア | 主要モデル | コスト (input/M) | 対象タスク |
|--------|-----------|-----------------|-----------|
| SIMPLE | NVIDIA Kimi K2.5 | $0.001 | 算術、定型処理 |
| MEDIUM | Grok Code Fast 1 | $1.50 | 要約、コード補助 |
| COMPLEX | Gemini 2.5 Pro | $10.00 | React実装、複雑な設計 |
| REASONING | Grok 4.1 Fast | $0.50 | 数学的証明、論理推論 |

特別ルールとして「推論マーカーが2個以上含まれる場合、信頼度0.97でREASONINGティアに振り分ける」という設定があり、推論タスクを$25/Mから$0.50/Mに削減できる。

**ブレンド平均コスト：$2.05/M**（Opus 4.5の$25/Mと比較して92%削減）

## 4つのルーティングプロファイル

```
/model auto     # デフォルト、品質とコストのバランス（74〜100%削減）
/model eco      # 最大コスト削減（95.9〜100%削減）
/model premium  # 最高品質のみ（Opus 4.5を使用）
/model free     # 無料ティアのみ（gpt-oss-120bを使用、コスト0）
```

`/model eco`切り替え後に「Write a React component」を投げると、DeepSeek（$0.28/$0.42）に自動ルーティングされ、Opus 4.5比98.3%削減になるという例が示されている。

## 最も革新的な部分：x402プロトコルによる決済

ClawRouterの最大の独自性は支払い方式だ。**APIキーを必要とせず、USDC（Base L2上）のウォレットが認証も兼ねる**。

```
Request → 402 (price: $0.003) → wallet signs USDC → retry → response
```

`x402`プロトコルというHTTP支払い拡張を使用し、リクエスト前に価格（402レスポンスのヘッダー）が確認でき、ウォレットが自動署名してリクエストを再送する。USDCは非カストディアル（自分の秘密鍵で管理）で、使った分だけ消費される。

```bash
# インストール
curl -fsSL https://blockrun.ai/ClawRouter-update | bash
openclaw gateway restart

# ウォレットに$5チャージで数千リクエストをカバー
```

この設計の含意は大きい。プロバイダーごとにAPIキーを取得・管理する手間がなく、$5程度のUSDCをチャージするだけで30種以上のモデルが使える。オープンなブロックチェーン支払いレールを使うことで、特定プロバイダーへの依存を回避している。

## Kimi K2.5：エージェントワークフロー向けの注目モデル

SIMPLEティアに採用されているMoonshot AIのKimi K2.5は、エージェントスウォームに最適化された仕様を持つ：

- **最大100並列エージェント**のコーディネート、実行速度4.5倍
- **200〜300回の連続ツール呼び出し**でもドリフトなく安定動作
- UIモックアップや動画からReactコードを生成（Vision-to-Code）
- エージェントベンチマークでOpus比76%低コスト

定型タスクをKimi K2.5（$0.001/M）に流し、複雑なタスクにだけOpus/Sonnetを使う構成が、ClawRouterの想定するユースケースだ。

## 独自評価

ClawRouterが注目される理由は3点に集約される。

**1. ゼロレイテンシ分類**：外部APIに依存しないローカル判定は、ルーター自体がパフォーマンスのボトルネックになる問題を回避する。

**2. APIキー不要の決済**：x402 + USDCという組み合わせは、プロバイダーアカウントを持たないユーザーでも30種以上のモデルにアクセスできる。これはLLMルーターのUX上の大きな障壁を除去する。

**3. エージェント特化の設計**：マルチステップタスクの自動検出、ツール有無の認識、コンテキストサイズフィルタリングなど、エージェントワークフローに特化した機能が標準で組み込まれている。

一方で注意点もある。USDCウォレット管理に慣れていない日本のユーザーには導入コストが高い可能性がある。また、$0.001/Mという超低価格のSIMPLEティアが実際の品質要件を満たすかはタスク次第だ。
