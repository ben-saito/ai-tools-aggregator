Cursor released version 2.7 on March 4, 2026. This update enables Cursor usage in JetBrains IDEs through the Agent Client Protocol (ACP).

## Cursor in JetBrains IDEs

Cursor is now available in IntelliJ IDEA, PyCharm, WebStorm, and other JetBrains IDEs through the [Agent Client Protocol](https://agentclientprotocol.com/) (ACP). Developers who rely on IntelliJ IDEA and other JetBrains IDEs for strong Java and multilanguage support can now use any frontier model with Cursor for agent-driven development.

## Key Features

### Multiple Model Selection

Different models are better suited for different kinds of tasks. With Cursor ACP, developers can explore and choose [frontier models](https://cursor.com/docs/models) from OpenAI, Anthropic, Google, and Cursor. Cursor's agent harness is custom-built for every model to optimize output quality and performance.

### Secure Codebase Indexing

Cursor uses secure codebase indexing and semantic search to understand large enterprise codebases. ACP combines these capabilities with deep code intelligence and tooling in JetBrains IDEs.

## Getting Started

Install the Cursor ACP directly in your JetBrains IDE from the ACP Registry, and authenticate with your existing Cursor account. The Cursor ACP is free for all users on paid plans.

## What's Next

The Cursor ACP is a foundation for deeper integrations with JetBrains. We're excited to bring agentic coding capabilities to more developers.
