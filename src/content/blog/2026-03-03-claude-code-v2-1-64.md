# Claude Code v2.1.64 Released — New Remote Control Feature and Performance Improvements

Claude Code v2.1.64 was released on March 2, 2026 (US time). This release includes the expansion of Remote Control functionality, performance improvements, and many new features.

## Key Changes

### New Remote Control Subcommand
The Remote Control feature, which provides local environment for external build environments, has been enhanced. More flexible operation is now possible when continuing sessions from mobile devices or browsers.

### BashTool Performance Improvement
Command execution is now faster by skipping the `-l` flag when using shell snapshots.

### Enhanced Auto-Save Functionality
The reliability of automatic saving has been improved, reducing the risk of losing important information during long sessions.

### Smarter Bash Command Suggestions
The command suggestion feature for complex commands has been improved, providing more accurate command recommendations.

### Memory Optimization for Multi-Agent Sessions
Memory usage when running multiple agents simultaneously has been optimized, enabling more efficient parallel processing.

### Security Fixes
- Enhanced security for statusLine/FileSuggestion hooks
- Security improvements for HTTP Hooks

### Windows Improvements
Multiple crash issues on Windows have been fixed, improving stability.

### Other Improvements
Listener-related memory leaks have been fixed, improving stability during extended use.

## How to Update

Claude Code automatically updates to the latest version. You can check the version with:

```bash
claude --version
```
