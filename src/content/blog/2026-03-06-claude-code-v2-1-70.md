# Claude Code v2.1.70 Released — New Slash Commands and Enhanced Memory Features

Claude Code v2.1.70 has been released in early March 2026 (US time). This release introduces new slash commands, significant memory enhancements, and various API and tooling improvements.

## Key Changes

### New Slash Commands

Two new bundled slash commands have been added:

- **/simplify**: Helps simplify code by refactoring complex code into simpler, more readable forms.
- **/batch**: Enables batch processing, allowing multiple tasks to be executed in one go.

### Enhanced Memory Features

Claude Code's memory capabilities have been significantly improved:

- **auto-memory**: Claude now automatically saves useful context to "auto-memory", which users can manage with the `/memory` command.
- **Multi-agent session improvements**: Memory usage in multi-agent sessions has been optimized for more efficient information sharing.

### Improved Code Handling

- **/copy command**: A new command that provides an interactive picker for code blocks, allowing users to select which code block to copy.

### Better Bash Integration

Smarter per-subcommand prefixes for compound bash commands have been introduced, improving command suggestions and helping developers use the terminal more efficiently.

### API and Tooling Improvements

- **HTTP hooks**: New capability to POST and receive JSON via HTTP, enabling flexible integration with external services.
- **ENABLE_CLAUDEAI_MCP_SERVERS=false**: A new environment variable to opt out of making claude.ai MCP servers available, providing more control over privacy and security settings.

### Model Command Enhancement

The **/model** command now displays the currently active model within the slash command menu, making it easier to track which model is in use.

### Stability and Performance

Multiple issues have been fixed:

- Fixed listener leaks in MCP OAuth flows
- Fixed memory leaks in the bash command prefix cache
- Fixed UI flicker issues
- Fixed various crashes on Windows and ARM64 Windows
- Other miscellaneous bug fixes

## Summary

Claude Code v2.1.70 is a feature-rich release aimed at improving developer productivity. The new `/simplify` and `/batch` slash commands, along with the enhanced auto-memory features, have the potential to significantly improve daily development workflows. The introduction of HTTP hooks also makes integration with external tools easier, which should contribute to ecosystem growth.

Users can update to the latest version using `claude --version`.
